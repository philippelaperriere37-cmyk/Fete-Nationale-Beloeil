# Brief du projet

## Objectif

Preparer une page Web statique pour la Fête nationale à Beloeil 2026.

## Direction retenue

Le projet utilise une structure simple en HTML, CSS et JavaScript, sans framework.

## Contraintes

- Garder le projet facile a modifier et a heberger.
- Prevoir des fichiers dedies aux informations extraites de Gmail.
- Ne pas figer le design final avant d'avoir les details de l'evenement.

## Structure approuvee

```text
/
  test 1.html
  README.md
  assets/
    images/
    icons/
    fonts/
    documents/
  css/
    main.css
  js/
    main.js
  content/
    event-notes.md
    source-emails.md
  docs/
    brief.md
```

## Prochaine etape

Valider la direction visuelle, ajouter les visuels officiels, puis raffiner la page pour publication.
