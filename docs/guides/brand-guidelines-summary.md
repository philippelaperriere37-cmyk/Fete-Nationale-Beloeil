# Synthèse - Normes graphiques Ville de Beloeil

Source : `assets/documents/guides/guide_normes_graphiques_vbeloeil.pdf`

## Palette officielle

Les couleurs du logo de la Ville de Beloeil sont :

| Usage | Pantone | RGB | Hex |
| --- | --- | --- | --- |
| Bleu principal | 7693 | 0, 79, 122 | `#004f79` |
| Jaune axe citoyen | 109 | 255, 209, 38 | `#ffd025` |
| Orange axe culture | 1585 | 243, 116, 46 | `#f3732e` |
| Vert axe développement | 361 | 67, 182, 73 | `#43b549` |
| Bleu pâle axe santé/nature | 299 | 0, 162, 221 | `#00a1e1` |

Pour cette page événement, l'axe culture peut soutenir la présence festive, et l'axe santé/nature peut soutenir les sections parc, plein air, eau et activités familiales. Le bleu principal doit rester la couleur institutionnelle dominante lorsque la Ville est mise de l'avant.

## Logo

- Le logo est composé de trois éléments indissociables : icône multicolore, nom `Beloeil` et positionnement `Forgée pour innover`.
- La version verticale en couleurs sur fond blanc est privilégiée.
- La version horizontale est réservée aux cas où l'espace rendrait la version verticale difficile à lire.
- Sur fond photo, utiliser une version approuvée qui conserve la lisibilité.
- Pour les tableaux de commanditaires, le logo de la Ville est isolé et utilisé en couleurs; les autres commanditaires peuvent être utilisés en noir.

## Zone de protection

Prévoir un dégagement minimal équivalent à la hauteur du `e` de `Beloeil` autour du logo. Aucun autre élément graphique ou texte ne doit entrer dans cette zone.

## Usages interdits

- Ne pas modifier les proportions du logo.
- Ne pas modifier l'espacement entre l'icône et le texte.
- Ne pas mettre le logo positif sur fond foncé ou sur fond de couleur.
- Ne pas placer le logo sur une photo texturée qui nuit à la lisibilité.
- Ne pas incliner le logo.
- Ne pas utiliser une version noire ou blanche seule sans approbation.

## Typographie

- Titres : Gotham Bold.
- Sous-titres : Gotham Bold en majuscules.
- Textes : Gotham Book.
- Titres de chapitre : Gotham Medium.
- Si Gotham n'est pas disponible, Calibri est le substitut prévu pour les documents Word.

## Application Web

Pour le site statique, utiliser une pile système qui se rapproche de Gotham si la police officielle n'est pas fournie :

```css
font-family: Gotham, Montserrat, Avenir, "Helvetica Neue", Arial, sans-serif;
```

Les titres doivent rester nets, institutionnels et lisibles. Les couleurs officielles peuvent être exposées en variables CSS pour assurer une cohérence entre la page, les boutons, les repères de programmation et les sections partenaires.
